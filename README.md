# Personal Scripts

A collection of utility scripts for Windows development and productivity.

## Configuration

To use these scripts globally from any terminal, you should add this directory to your system's `PATH`.

### Adding to PATH (Windows)

1.  Copy the full path to this directory (e.g., `E:\work\personal\scripts`).
2.  Open **Start Search**, type **"env"**, and select **Edit the system environment variables**.
3.  Click the **Environment Variables...** button.
4.  Under **User variables**, select the **Path** variable and click **Edit...**.
5.  Click **New** and paste the path to this directory.
6.  Click **OK** on all windows to save.
7.  **Restart** your terminal or IDE for the changes to take effect.

> [!TIP]
> You may also want to ensure that `.PY` is included in your `PATHEXT` environment variable if you want to run Python scripts without typing the `.py` extension.

## Scripts

### 📂 Directory Tree Generator (`directorytree.py`)
Creates a beautiful UTF-8 tree diagram of any directory structure.

**Usage:**
```powershell
# Print to console
python directorytree.py "C:\Path\To\Folder"

# Save to a file
python directorytree.py "C:\Path\To\Folder" -o tree.txt

# Include hidden files
python directorytree.py "C:\Path\To\Folder" --show-hidden
```
